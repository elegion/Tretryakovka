# Tretyakovka

## Description

Custom and customizable iOS image gallery customized similarly to Photo app.

## Installation

With CocoaPods:

```
source 'https://github.com/elegion/ios-podspecs.git'
source 'https://github.com/CocoaPods/Specs.git'

pod 'Tretyakovka'
```
